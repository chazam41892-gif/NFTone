    Watermark assignment

    Each authorized wallet gets a perceptually identical but bit-distinct render. Payload layers below are illustrative; real payloads are HSM-wrapped.

    Payload structure

    24b release_id 0x8f3a92
    32b wallet_index 0xb1c7d4e6
    4b version 0x1
    4b crc 0xa

    The on-chain side never sees this payload. The wallet address is never embedded — only an opaque index into the controlled mapping table.
